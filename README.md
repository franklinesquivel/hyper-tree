# Hyper Tree

### Setting up project

> Install dependencies

```console
yarn install
```

or simply:

```console
yarn
```

> Build project TS files

```console
yarn build
```

> Build project TS files in watch mode

```console
yarn dev
```

> Run build

```console
yarn start
```

> Lint project

```console
yarn lint
```

> Lint and fix project

```console
yarn lint:fix
```
